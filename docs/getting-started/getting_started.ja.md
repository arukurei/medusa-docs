# はじめる

![Title.png](../assets/images/Title.png)

**Medusa** は、ダイナミックなUIグラフ、スキルツリー、ノードベースのウェブを構築するための高性能なGodotプラグインです。

標準的な静的UIソリューションとは異なり、Medusaにはデータ指向の物理エンジン、進行状況の追跡、最適化されたレンダリング機能が組み込まれています。このアドオンは、開発者にカスタムControlノード（`Graph`、`Atom`、`GraphBranch`など）と高度な描画ツールのセットを提供し、複雑でインタラクティブな物理ベースのウェブを簡単に作成できるようにします。

---
## インストール
### - **Godot Asset Library** <small>(推奨)</small>

プラグインはアセットライブラリに登録されており、そこからプロジェクトに直接インストールすることができます。

これを行うには、Godotエディタウィンドウの上部にある「AssetLib」タブに移動して **Medusa** を検索するか、公式の **Godot Asset Library** ウェブサイトを使用します。

---
### - **GitHub**

プロジェクトのオープンソースコードと利用可能なすべてのリリースは、**[GitHub](https://github.com/arukurei/Medusa)** プラットフォーム上でホストされています。

手動で **Medusa** をインストールしてみましょう：

- 「Releases」セクションから希望するバージョンのアドオンのZIPファイルをダウンロードします。
- プラグインフォルダを含むアーカイブを解凍します。
- 解凍したフォルダをプロジェクトの `addons` ディレクトリに移動します。

正しくインストールされているか確認してみましょう：

- `res://addons/medusa` フォルダへのパスが存在することを確認します。
- プロジェクト > プロジェクト設定 に移動します。
- **プラグイン (Plugins)** タブをクリックします。
- **Medusa** の横にある有効化ボタンにチェックを入れます。
