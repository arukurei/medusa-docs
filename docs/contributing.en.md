# Contribution

## How to contribute

There are different ways to contribute to "Medusa". You can be a developer, a designer, a writer, or a user who wants to help improve the project. Here are a few ways you can contribute:

---
## 🐛 Bug Report

If you experience any broken functionality, problem, or any other undesirable effect, please create a [Bug Report](https://github.com/arukurei/Medusa/issues/new/choose).

---
## ✨ Feature Request

Not all possible features have been planned or considered. If there is something you would like to see, please submit a [Feature Proposal](https://github.com/arukurei/Medusa/issues/new/choose).

---
## 🧰 Technical Contribution

If you're tech-savvy and want to help make the add-on better, code contributions are very welcome!

### Major Changes

**Examples:**

- New features and ideas
- Improving/refactoring old functionality

Please open a [Feature Proposal](https://github.com/arukurei/Medusa/issues/new/choose) or [Bug Report](https://github.com/arukurei/Medusa/issues/new/choose) before you start working on any major changes.

If a bug report or feature proposal already exists and has been stagnant for a long time, feel free to leave a comment and try to implement it!

### Minor Changes

**Examples:**

- Small code changes that affect specific area(s)
- Bug fixes
- Typos and formatting

Feel free to send pull requests or suggestions for any minor changes. Even the smallest things help a lot!

---
## 📃 Documentation

You can also help develop the documentation: suggest fixes for inaccuracies and typos, or suggest new pages and details.

Raise any questions on the [Ask a Question](https://github.com/arukurei/medusa-docs/issues/new/choose) page.

---
## ❤️ Donations

If you like Medusa and want to support its development, you can donate to the project via [PayPal](https://www.paypal.com/donate/?hosted_button_id=LF5SHGQDXK2PG) or [ItchIO](https://alkrei.itch.io/medusa). Your support is greatly appreciated and helps keep the project going!
